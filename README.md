## Compatibility matrix

|               | Bass | Dory | Lenok | Sparrow | Smelt | Sturgeon |
| ------------- | ---- | ---- | ------- | ------- | ----- | -------- |
| XA-2         | ❔️    | ❔    | ❔     | ❔       | ❔️     | ❔        |
| X 10 II       | ❗    | ❔    | ❔       | ❔       | ❔     | ❔        |


| Symbol | meaning                  |
| ------ | ------------------------ |
|  ✔️     | Working fine             |
|  ❌     | Doesn't connect          |
|  ❗     | Loses connection        |
|  ❔     | No information available |
